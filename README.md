# Portfólio — Matheus Carvalho

Portfólio pessoal desenvolvido com Next.js 16, apresentando minha trajetória, projetos e habilidades como desenvolvedor Backend Java em formação.

## Sobre o projeto

Design premium dark com acento dourado, construído do zero com foco em performance e animações fluidas. Cada seção foi projetada para comunicar identidade técnica e profissional de forma clara e elegante.

## Stack

| Camada | Tecnologia |
|--------|-----------|
| Framework | Next.js 16 (App Router) |
| Linguagem | TypeScript |
| Estilização | Tailwind CSS v4 |
| Animações | Framer Motion 12 |
| Efeito visual | THREE.js — WebGL shader |

## Seções

- **Hero** — Shader WebGL com efeito de aberração cromática e entrada animada
- **Sobre** — Perfil profissional e pilares de valor
- **Stack** — Tecnologias organizadas por categoria (Backend, Cloud, Ferramentas...)
- **Projetos** — Cards com Canvas 2D animado e links para repositórios
- **Certificações** — Certificações concluídas e próximas em andamento
- **Experiência Profissional** — e.Sales, GRÉGORA e RENAPSI com tech tags
- **Contato** — Formulário e links diretos

## Como executar localmente

**Pré-requisito:** Node.js 18 ou superior.

```bash
# 1. Clone o repositório
git clone https://github.com/Matheusovc/portfolio.git
cd portfolio

# 2. Instale as dependências
npm install

# 3. Inicie o servidor de desenvolvimento
npm run dev
```

Acesse **[http://localhost:3000](http://localhost:3000)** no navegador.
